# Quark
NEUTRINOで出力した音響データを調整するツール

![image](https://user-images.githubusercontent.com/6949910/230702495-5c5b265f-131f-45f9-b310-caf27316f050.png)

## 動作について
仮実装中です。  
編集機能は実装予定ですが、現時点ではNEUTRINOで出力したデータを眺める事くらいしかできません。  
プロジェクトデータ周りの画面制御がまだ整備できていないため、少し挙動が不安定です。  

## 使い方
0. 事前準備
  - MusicXMLファイルを用意しておく。  
      現時点では、一部の音符に歌詞が入っていないなど、NEUTRINOでエラーが吐かれるようなものは扱えません。  
      また、NEUTRINOの挙動により、有音が1小節目の先頭から始まっている場合、出力データが1小節ほど後ろにずれる事があります。  
  - 設定画面でNEUTRINO(v1)の実行ディレクトリを設定しておく。  
1. ファイルメニューから新規プロジェクトを作成する。
2. プロジェクトメニューからMusicXML取り込みを行う。
3. NEUTRINOの処理が正常終了すれば、上記画像のようにピアノロール上にデータが表示されます。

## ビルドについて
.NET 8 SDK と C# 12.0 、WPF、WinForms が使用できる環境が必要です。
※ Visual Studio 2022を推奨、その他の環境は未確認です。

このリポジトリをローカル環境にクローンし、Visual Studioで`Quark.Prototype`プロジェクトをビルド実行していただければ動作するかと思います。

## 実装について
プロトタイプとしてWindowsで動くものを作っていますが、将来的にはクロスプラットフォーム対応を考えています。
描画部分は後々の移植性を考慮してSkiaを用いた実装としています。
